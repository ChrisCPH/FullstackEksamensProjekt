# Eksamens Projekt
**Gruppenavn:** Team Fullstack

# Installation
cd to ``front-end`` and ``back-end`` and run the following command:
* ```npm install```

# Starting The Servers

## Front-end
```npm run dev```

## Back-end
Create config.env file with the following info
- ```PORT=```
- ```DATABASE_DEV=```
- ```DATABASE_PASSWORD=```

To start server:
```npm run start:nodemon```
